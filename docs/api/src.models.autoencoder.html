class src.models.autoencoder.Model1(input_shape)[source]¶

Bases: object

A first auto-encoder model.

The model is a convolutional model, adapted to the problem. The inputs have originally two dimensions, the timestack size. The shape of the output of the encoder is the reduce dimensions (compression).

The encoder is composed of three convolutional layers, with non-square max-pooling after each convolutional layer. The decoder is classically composed of three convolutional layers too, in the opposite direction.

Example

>>> autoencoder = Model1((400, 200, 1))
>>> print("Encoder summary.")
>>> print(autoencoder.encoder().summary())
>>> print("Auto-__cnn summary.")
>>> print(autoencoder.autoencoder().summary())
